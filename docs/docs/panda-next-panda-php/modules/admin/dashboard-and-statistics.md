# 仪表盘与统计

## 功能目标

让后台快速感知站点当前运行状况和关键业务变化。

## 参与角色

- 管理员
- 运营人员

## 入口

- 后台仪表盘首页

## 前台功能

- 无

## 后台功能

- 查看系统信息
- 查看站点统计
- 查看最新用户
- 查看最新种子

## 状态定义

- 即时统计
- 最近变化
- 历史趋势（如后续扩展）

## 状态流转

1. 后台请求统计接口
2. 系统聚合当前状态
3. 返回仪表盘卡片和列表数据

## 规则与校验

- 指标口径必须统一
- 最新用户和最新种子要与真实数据一致
- 系统信息不能暴露不应显示的敏感内容

## 异常与边界

- 统计计算失败
- 数据延迟
- 仪表盘局部模块无数据

## 数据结果

- 系统信息
- 统计聚合结果
- 最新数据列表

## 验收标准

- 仪表盘可稳定展示关键站点状态
- 指标与后台查询口径一致
- 局部异常不影响整体页面可用性
