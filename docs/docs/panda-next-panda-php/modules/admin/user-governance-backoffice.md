# 用户治理后台

## 功能目标

集中处理用户列表、详情、限制、恢复和安全治理动作。

## 参与角色

- 管理员
- 被管理用户

## 入口

- 用户列表页
- 用户详情页
- 用户处理操作区

## 前台功能

- 无直接前台页面，前台只消费处理结果

## 后台功能

- 用户 CRUD
- 禁用/启用
- 重置密码
- 修改字段值
- 查看邀请信息
- 查看管理备注
- 移除二步验证

## 状态定义

- 可管理
- 已限制
- 已恢复

## 状态流转

1. 管理员定位目标用户
2. 执行治理动作
3. 写入处理结果和审计记录
4. 前台权限和状态立即改变

## 规则与校验

- 管理动作必须校验操作者权限
- 敏感操作必须留痕
- 字段增减动作必须限定可操作字段

## 异常与边界

- 用户不存在
- 重复处理
- 越权管理

## 数据结果

- 用户状态更新
- 安全字段更新
- 治理审计记录

## 验收标准

- 后台能稳定完成核心用户治理动作
- 用户前台状态随后台动作即时变化
- 敏感处理动作都可追溯
