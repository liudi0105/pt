# 种子列表与详情

## 功能目标

提供资源浏览主入口，让用户能够发现、查看并进入下载流程。

## 参与角色

- 游客或登录用户（取决于站点策略）
- 发布者
- 管理员

## 入口

- 种子列表页
- 种子详情页
- 用户中心的种子关联页

## 前台功能

- 浏览列表
- 分页
- 排序
- 查看详情
- 查看文件列表
- 查看评论、感谢和书签状态
- 进入下载

## 后台功能

- 按条件查询资源
- 查看资源完整信息

## 状态定义

- 可见
- 不可见
- 待审核
- 已拒绝
- 已删除/下架

## 状态流转

1. 种子创建后进入待审核或可见状态
2. 审核通过后进入公开列表
3. 下架或删除后从公开结果中移除

## 规则与校验

- 未公开资源不能进入普通用户列表
- 详情页显示内容受资源状态影响
- 用户权限影响是否可下载、评论或感谢

## 异常与边界

- 资源不存在
- 资源已删除
- 资源存在但无权查看

## 数据结果

- 列表结果集
- 详情聚合视图
- 资源展示统计输入

## 验收标准

- 列表与详情的可见性规则一致
- 已审核和未审核资源不会混淆展示
- 详情页能正确聚合文件、评论和互动信息
