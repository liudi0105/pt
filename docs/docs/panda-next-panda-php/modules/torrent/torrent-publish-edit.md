# 种子发布与编辑

## 功能目标

让资源以规范方式进入站点，并允许在规则范围内修订元数据和附件信息。

## 参与角色

- 发布者
- 管理员

## 入口

- 发布页
- 编辑页
- API 发布与更新接口

## 前台功能

- 新建种子
- 编辑种子
- 上传文件和附件
- 填写分类、标签、来源、编码、字幕等元数据

## 后台功能

- 查看发布内容
- 在必要时修正资源信息

## 状态定义

- 草稿/待提交
- 待审核
- 已公开
- 已拒绝
- 不可编辑

## 状态流转

1. 发布者提交资源
2. 系统校验元数据完整性
3. 根据站点规则决定直发或待审核
4. 审核通过后公开
5. 审核拒绝后返回待修改或终止

## 规则与校验

- 必填元数据必须完整
- 非法状态下禁止编辑或删除
- 文件和附件必须与资源主体绑定
- 已公开资源的可编辑范围需要受控

## 异常与边界

- 上传内容不完整
- 元数据缺失
- 用户无发布权限
- 审核中资源重复提交

## 数据结果

- 种子主体
- 文件列表
- 元数据关系
- 附件引用

## 验收标准

- 发布、编辑、审核前状态切换完整
- 元数据与附件引用不会脱离主体资源
- 权限不足用户无法发布或修改受限资源
