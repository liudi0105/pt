# 用户主档案

## 功能目标

定义用户的主身份资料、偏好资料和展示资料，作为用户中心和后台管理的共同来源。

## 参与角色

- 普通用户
- 管理员

## 入口

- 用户中心
- 管理后台用户详情页
- 当前用户信息接口

## 前台功能

- 查看个人基础资料
- 查看个人统计摘要
- 查看等级和状态
- 查看个人展示偏好

## 后台功能

- 新建用户
- 编辑用户
- 删除用户
- 读取用户基础资料
- 读取管理备注

## 状态定义

- 资料完整
- 资料缺失
- 可编辑
- 只读

## 状态流转

1. 用户创建时生成主档案
2. 系统补充默认偏好和默认等级
3. 用户或管理员修改资料
4. 前台与后台读取同一份主事实

## 规则与校验

- 用户身份字段与展示字段要分层
- 管理备注不应混入用户公开资料
- 统计字段不可被前台任意修改
- 删除用户必须定义软删除或硬删除策略

## 异常与边界

- 用户不存在
- 用户资料部分缺失
- 删除后历史关联数据仍需可追溯

## 数据结果

- 用户主表数据
- 用户扩展资料
- 用户展示偏好
- 用户审计关联入口

## 验收标准

- 当前用户信息与后台用户详情口径一致
- 主档案、扩展档案和审计信息分层明确
- 用户删除策略可说明历史数据去向
