# 评论、感谢与书签

## 功能目标

承载资源互动行为，形成活跃度和内容反馈输入。

## 参与角色

- 普通用户
- 发布者
- 管理员

## 入口

- 种子详情页
- 用户中心互动列表

## 前台功能

- 发表评论
- 编辑和删除评论（按权限）
- 表达感谢
- 添加书签/收藏

## 后台功能

- 管理评论内容
- 处理违规互动

## 状态定义

- 评论已发布
- 评论已删除/隐藏
- 已感谢
- 已收藏

## 状态流转

1. 用户进入详情页
2. 执行评论、感谢、书签动作
3. 前台立即更新互动状态
4. 后台可对违规内容进行治理

## 规则与校验

- 评论必须绑定业务对象
- 感谢和书签通常应避免重复写入
- 删除评论需定义是否物理删除或逻辑隐藏

## 异常与边界

- 对不存在对象评论
- 重复感谢
- 重复书签
- 无权编辑他人评论

## 数据结果

- 评论记录
- 感谢记录
- 书签记录
- 热度和活跃度输入

## 验收标准

- 评论、感谢、书签在详情页行为一致
- 重复动作得到稳定处理
- 后台能准确治理违规互动内容
