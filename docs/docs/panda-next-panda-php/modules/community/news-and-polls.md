# 新闻与投票

## 功能目标

承载站点公告、活动传播和用户意见收集能力。

## 参与角色

- 普通用户
- 运营人员
- 管理员

## 入口

- 新闻列表页
- 新闻详情页
- 投票列表页
- 投票详情页

## 前台功能

- 查看新闻
- 查看最新新闻
- 查看投票
- 参与投票
- 查看投票结果（依规则）

## 后台功能

- 创建、编辑、删除新闻
- 创建、编辑、删除投票
- 维护投票选项

## 状态定义

- 已发布
- 未发布
- 投票进行中
- 投票已结束

## 状态流转

1. 运营创建新闻或投票
2. 内容发布后进入前台
3. 投票开放期间用户可投票
4. 投票截止后只保留结果展示

## 规则与校验

- 投票是否允许重复投必须明确
- 已结束投票禁止继续投票
- 新闻与投票的发布时间必须可控

## 异常与边界

- 投票选项不存在
- 重复投票
- 已下线新闻仍被访问

## 数据结果

- 新闻内容
- 投票主题与选项
- 用户投票结果

## 验收标准

- 新闻与投票前后台闭环完整
- 投票规则可被严格执行
- 最新内容接口与详情页口径一致
