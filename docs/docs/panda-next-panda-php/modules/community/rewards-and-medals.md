# 奖励与勋章

## 功能目标

承载长期激励和身份展示能力，使站点能够区分荣誉、活动、权益和积分变化。

## 参与角色

- 普通用户
- 运营人员
- 管理员

## 入口

- 用户中心奖励记录页
- 勋章展示页
- 后台勋章管理页

## 前台功能

- 查看奖励记录
- 查看勋章列表和已拥有勋章
- 查看奖励来源

## 后台功能

- 定义勋章
- 发放勋章
- 管理用户勋章关系
- 记录奖励变动

## 状态定义

- 勋章未拥有
- 勋章已拥有
- 奖励待发放
- 奖励已发放

## 状态流转

1. 系统或管理员触发奖励
2. 写入奖励日志或勋章关系
3. 前台展示新结果
4. 如有时效，按规则失效或下架

## 规则与校验

- 奖励必须有来源
- 勋章展示与权益属性需要区分
- 用户勋章不能无来源写入

## 异常与边界

- 重复发放
- 过期勋章仍展示
- 奖励来源丢失

## 数据结果

- 奖励日志
- 勋章定义
- 用户勋章关系

## 验收标准

- 奖励和勋章的发放、展示、追踪完整可用
- 用户能清楚看到来源和当前持有结果
- 后台可追溯每次发放动作
